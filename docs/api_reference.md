# API Reference

Primary modules:

- `orderflowkit.feeders` for historical bars and trades.
- `orderflowkit.streams` for public L2 streams.
- `orderflowkit.book` for local order-book reconstruction.
- `orderflowkit.metrics` for microstructure estimators.
- `orderflowkit.features` for ML-ready feature tables.
- `orderflowkit.labels` for future-return and classification labels.
- `orderflowkit.replay` for deterministic event replay.
- `orderflowkit.cli` for the `ofk` command.
